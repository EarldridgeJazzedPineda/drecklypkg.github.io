Portable Package Management

pkgsrc is a framework for building over 14,000 open source software packages. It is the native package manager on SmartOS, NetBSD, and Minix, and is portable across 22 different operating systems. Use one package manager across all of your systems!

Joyent provide binary packages for SmartOS/illumos, Mac OS X, and Linux.
